# README

## USERS
### See all users data
`http://localhost:3000/users/`
### See user data
`http://localhost:3000/users/<user_id>`

## FOLLOW and UNFOLLOW
### Follow user
`http://localhost:3000/users/<user_id>/follow/<target_id>`
### Unfollow user
`http://localhost:3000/users/<user_id>/follow/<target_id>`
